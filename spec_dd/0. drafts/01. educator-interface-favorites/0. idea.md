Allow the user to "favorite" specific pages in the educator interface by clicking a star icon  on the page (next to the heading)

Favorites then get displayed in the left hand navigation, at the top.

Note that we will create other interfaces for different types of users, so we need something that works outside of the educator interface. It should be in the panels app.

class Favorite(site aware model):
    user
    interface_name (eg educator interface)
    page


As soon as a user favorites something, it should appear in the navigation. As soon as they unfavorite something, it should disappear.
