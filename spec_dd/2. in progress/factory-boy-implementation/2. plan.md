## Phase 1: Base Factory and Core Factories

### Task 1.1 — Create `SiteAwareFactory` base class

**File**: `freedom_ls/site_aware_models/factories.py`

**Skill**: `tdd_implement`

Create the base factory that all site-aware factories inherit from:

```python
import factory
from django.contrib.sites.models import Site
from freedom_ls.site_aware_models.models import _thread_locals

def _get_current_site():
    from django.contrib.sites.shortcuts import get_current_site
    request = getattr(_thread_locals, "request", None)
    if request:
        return get_current_site(request)
    return None

class SiteAwareFactory(factory.django.DjangoModelFactory):
    site = factory.LazyFunction(_get_current_site)
```

**Tests**: Verify that `SiteAwareFactory` subclasses correctly pick up site from `mock_site_context`. Test that site can be overridden explicitly.

### Task 1.2 — Create `accounts/factories.py`

**File**: `freedom_ls/accounts/factories.py`

**Skill**: `tdd_implement`

Use `PostGenerationMethodCall` to handle password hashing — this is the standard factory_boy pattern for Django users:

```python
class UserFactory(SiteAwareFactory):
    class Meta:
        model = User

    email = factory.Sequence(lambda n: f"user{n}@example.com")
    @factory.post_generation
    def password(obj, create, extracted, **kwargs):
        """Set password equal to the user's email address."""
        obj.set_password(extracted or obj.email)
        if create:
            obj.save(update_fields=["password"])
    is_active = True
    is_staff = False

    class Params:
        staff = factory.Trait(is_staff=True)
        superuser = factory.Trait(is_staff=True, is_superuser=True)
```

**Tests**: Verify `UserFactory()` creates a user with hashed password. Verify `UserFactory(staff=True)` sets `is_staff=True`. Verify `UserFactory(superuser=True)` sets both flags. Verify site is set from `mock_site_context`.
