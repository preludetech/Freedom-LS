# Plan: Bugfix — TOC expand/collapse broken on course detail pages

## Summary

The `coursePart` Alpine.js component (which provides `toggleExpanded`) is only loaded on `course_home.html` via the `extra_alpine_components` block. All course detail pages (`course_topic.html`, `course_form.html`, `course_form_page.html`, `course_finish.html`, `course_form_complete.html`) extend `_course_base.html`, which does not override this block. The fix is to add the block override to `_course_base.html`.

## Skills to use

- **fls:testing** — for writing pytest tests (TDD)
- **fls:template** — for editing `_course_base.html` with correct template conventions
- **fls:alpine-js** — for understanding Alpine.js component loading patterns

## Tasks

### Task 1: Write a failing test

**Skill:** `fls:testing`

Write a test that verifies the Alpine components script tag is present in the rendered HTML of course detail pages. Use Django's test client to render a course topic page and assert that `alpine-components.js` appears in the response content.

**File to edit:** Create test in `freedom_ls/student_interface/tests/` (find the appropriate existing test file for template/view tests, or add to an existing test module).

**Pseudocode:**
```python
def test_course_detail_pages_load_alpine_components_script(self):
    # Set up: create a course with a topic
    # GET the course topic detail page
    response = self.client.get(reverse("student_interface:course_topic", args=[course.slug, topic.slug]))
    assert response.status_code == 200
    assert "alpine-components.js" in response.content.decode()
```

Run with `uv run pytest` and confirm the test fails.

### Task 2: Add the `extra_alpine_components` block to `_course_base.html`

**Skills:** `fls:template`, `fls:alpine-js`

**File to edit:** `freedom_ls/student_interface/templates/student_interface/_course_base.html`

Add `{% load static %}` at the top (after the `{% extends %}` tag) and the `extra_alpine_components` block, following the same pattern as `course_home.html`:

```django
{% load static %}

{% block extra_alpine_components %}
    <script defer src="{% static 'student_interface/js/alpine-components.js' %}"></script>
{% endblock extra_alpine_components %}
```

Run `uv run pytest` and confirm the test from Task 1 now passes.

### Task 3: Verify no regression on course_home.html

Ensure the existing `extra_alpine_components` block in `course_home.html` still works. The existing test suite should cover this — run the full test suite with `uv run pytest` and confirm all tests pass.
