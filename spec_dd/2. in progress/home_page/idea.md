A user who is not logged in should be able to see what the platform is about before logging in/making an account.

Current behaviour: users get redirected to login page if they try to access things

Wanted behavior for unauthenticated users:
- users should be able to see something like the "dashboard". Show a few courses, allow navigation. Reuse as much of the dashboard as possible, maybe just hide some sections for unauthenticated learners
- users should be able to navigate to the course catalogue
- learners should be able to see a course details page and see the CTA there. If they choose to apply to a course or enroll in a free course or whatever, they should be asked to login/create an account.
