Once a user has registered for a course, there is no obvious way to get back to the course details page. The course details page is helpful if, for example, they want to share a link with a friend or organisation.

We need the following:

1. Dashboard page
Course cards default behavior is correct
Add a mechanism that allows the user to link to the details page
UX: Have a "3 dots" menu button to the card, bottom left. Clicking on it will open up a menu with a "details" link.

2. Course index page:
Each course list item should have a clear way of accessing the details page. Aim for a consistent UX experience.

3. Course player:
Adjust the first breadcrumb so it links to the details page. If other things should be adjusted, do that too.
