When moving around the educator interface, it's easy to lose track of where we are.

## Breadcrumbs
Implement breadcrumbs at the top of the screen.
Eg if we click on "Cohorts" then select a specific cohort, show this in the breadcrumbs.

## Left panel navigation highlight
If the user selects something from the left hand panel, then highlight the active link.

## Left panel navigation dropdowns
If the user navigates to a specific instance (eg a specific cohort) then display that cohort in the left hand navigation panel. Like this:

```
- Cohorts >
    - selected cohort
- Users
- Courses
```
