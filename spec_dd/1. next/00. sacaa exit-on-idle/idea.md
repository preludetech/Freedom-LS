During exam type forms, we need to track idle time. If a learner is doing an exam and they are not interacting with their computer, then we need to auto-submit the exam.

The idle time for an exam must be pre-configured and it must be communicated to the learner ahead of time so that they don't get logged out by mistake. If the learner is idle for a specified time limit, for example, two minutes and thirty seconds, then they will get a message on the screen saying that they were idle and the form will be submitted.

This should be configurable. All exam type forms should have the same idle time limits.
