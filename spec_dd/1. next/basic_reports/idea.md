We need the system to email reports about student progress to people.

At a minimum we'll want a report that shows information about active learners and cohorts
Columns:
- email
- student name
- cohorts
- course progress % (might have multiple courses)
- last access date
- for each test
    - number of failures
    - last accessed date
    - passed



- Super admin should be able to set up system-wise defaults
- Individual staff members can override

We want to make sure whatever we build can be extended on later. It might make sense to build this into multiple apps, one for managing notifications in general, something for report generation.
