We need to make use of outbound  webhooks to notify external platforms of different things happening on our platform. But some of these platforms don't implement webhooks in the usual sense, they have different types of endpoints.

For example: If we wanted to use Brevo to send a series of emails when a new user registers (user.registered webhook event) on the platform then we need to plug into the api defined here: https://developers.brevo.com/docs/track-custom-events-rest

So we need to have a way to specify:
- headers
- data structures
- secrets

We need to allow a user to override and configure behavior through the admin panel.
