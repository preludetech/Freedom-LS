## Task 1: Test for `UserManager.get_queryset()` respecting `FORCE_SITE_NAME`

**Skill:** `testing` (TDD)

Write a failing test in `freedom_ls/accounts/tests/test_user_manager.py` (new file).

Setup:
- Create two sites: `ForcedSite` (domain `forced.example.com`) and `DomainSite` (domain `testserver` — the default RequestFactory domain)
- Create a user on each site
- Use `override_settings(FORCE_SITE_NAME="ForcedSite")`
- Set `_thread_locals.request` to a real `RequestFactory().get("/")` request (domain will be `testserver`)

Assert:
- `User.objects.all()` returns only the ForcedSite user (not the DomainSite user)
- This proves UserManager uses the forced site, not the request domain

Use the pattern from `test_get_cached_site.py` — use `RequestFactory`, `override_settings`, and direct manipulation of `_thread_locals`. Clean up `_thread_locals.request` in teardown.

```python
# Pseudocode
@pytest.mark.django_db
def test_user_manager_respects_force_site_name():
    forced_site = Site.objects.create(domain="forced.example.com", name="ForcedSite")
    domain_site = Site.objects.create(domain="testserver", name="DomainSite")

    # Create users directly via ORM (bypass manager filtering)
    User = get_user_model()
    User.objects.create_user(...)  # on forced_site
    User.objects.create_user(...)  # on domain_site
    # NOTE: create_user goes through UserManager which filters by site,
    # so we need to create users without the manager filter active
    # (i.e., with no request in _thread_locals, or use the base manager)

    request = RequestFactory().get("/")  # domain = testserver
    _thread_locals.request = request

    with override_settings(FORCE_SITE_NAME="ForcedSite"):
        # Clear any cached site on the request
        users = list(User.objects.all())

    # Cleanup _thread_locals
    del _thread_locals.request

    assert len(users) == 1
    assert users[0].site == forced_site
```

**Run test — expect FAILURE** (Red phase).

```python
# Pseudocode
@pytest.mark.django_db
def test_is_open_for_signup_respects_force_site_name(settings):
    forced_site = Site.objects.create(domain="forced.example.com", name="ForcedSite")
    Site.objects.create(domain="testserver", name="DomainSite")

    SiteSignupPolicy.objects.create(site=forced_site, allow_signups=False)
    settings.ALLOW_SIGN_UPS = True  # global default

    request = RequestFactory().get("/")  # domain = testserver

    with override_settings(FORCE_SITE_NAME="ForcedSite"):
        result = AccountAdapter().is_open_for_signup(request)

    assert result is False  # Should use ForcedSite's policy, not DomainSite
```
