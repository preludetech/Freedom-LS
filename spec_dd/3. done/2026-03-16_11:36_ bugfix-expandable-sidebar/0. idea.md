# Change base_interface behavior

freedom_ls/base/templates/_base_interface.html

Desired behavior:

All devices:
- The expand and collapse buttons must be aligned vertically. It should look like one button (or maybe it shoud be one button and the icon can change )

Phones:
- hide the side panel menu by default
- expanding the panel should cause a drawer to slide out over the main content. The main content should be obscured
- clicking on a link in the side panel should navigate AND hide the side panel again

Tablets and large screens:
- show the side panel by default
- opening the side panel should cause a drawer to slide out and compress the main content (dont overlay or obscure)
- clicking on a link in the side panel should navigate and keep the side panel open
