## Task 1: Pre-migration validation

Write a data migration that asserts no `User` has multiple `Student` records. This protects against data corruption before the FK swap.

Create migration file in `freedom_ls/student_management/migrations/`.

```python
def forward(apps, schema_editor):
    from django.db import connection
    with connection.cursor() as cursor:
        cursor.execute("""
            SELECT user_id, COUNT(*)
            FROM freedom_ls_student_management_student
            GROUP BY user_id HAVING COUNT(*) > 1
        """)
        duplicates = cursor.fetchall()
        if duplicates:
            raise Exception(f"Duplicate Student records for user_ids: {[r[0] for r in duplicates]}")
```

### `freedom_ls/student_management/utils.py` (create if needed)

Move `Student.get_course_registrations()` → `get_course_registrations(user: User) -> list[Course]`:

```python
def get_course_registrations(user: User) -> list[Course]:
    """Return all courses a user is registered for (direct + cohort)."""
    direct = UserCourseRegistration.objects.filter(
        user=user, is_active=True
    ).values_list("collection", flat=True)

    cohort = CohortCourseRegistration.objects.filter(
        cohort__cohortmembership__user=user, is_active=True
    ).values_list("collection", flat=True)

    return list(Course.objects.filter(Q(pk__in=direct) | Q(pk__in=cohort)).distinct())
```
