# Quiz User Stories

## Student Experience

### Taking Quizzes


- As a student, I can see immediate feedback on my quiz results (if enabled)
- As a student, I can retake a quiz if multiple attempts are allowed
- As a student, I can see my quiz history and previous scores

### Quiz Progress

- As a student, I can see a timer if the quiz is time-limited
- As a student, I receive a warning before time expires on timed quizzes

## Instructor/Admin Experience

### Quiz Creation
- As an instructor, I can create a quiz with multiple questions
- As an instructor, I can add different question types (multiple choice, true/false, short answer, essay)
- As an instructor, I can set correct answers and point values for each question
- As an instructor, I can reorder questions within a quiz

### Quiz Configuration
- As an instructor, I can set a time limit for quiz completion
- As an instructor, I can configure the number of allowed attempts
- As an instructor, I can set passing score thresholds
- As an instructor, I can choose to randomize question order
- As an instructor, I can show/hide correct answers after submission
- As an instructor, I can enable/disable immediate feedback

### Quiz Management
- As an instructor, I can view all student submissions and scores
- As an instructor, I can manually grade subjective questions (short answer, essay)
- As an instructor, I can export quiz results to CSV
- As an instructor, I can see quiz analytics (average score, completion rate, difficult questions)

## Quiz Types
- As a course designer, I can create formative quizzes (practice, no grade impact)
- As a course designer, I can create summative quizzes (graded assessments)
- As a course designer, I can create surveys (no correct answers)

## Grading & Tracking
- As a student, I can see my quiz grades in my course progress
- As a student, I can see which quizzes are required for course completion
- As the system, I auto-grade objective questions (multiple choice, true/false)
- As the system, I track quiz completion as part of overall course progress
