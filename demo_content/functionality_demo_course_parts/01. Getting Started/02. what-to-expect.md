# What to Expect

This course has three parts, each demonstrating different ways to use course parts:

- **Part 1: Getting Started** — Contains only topics, showing the simplest use of parts.
- **Part 2: Core Concepts** — Contains topics and a quiz, showing that parts can mix content types.
- **Part 3: Wrapping Up** — Contains a topic and a survey, demonstrating a different form type within a part.
