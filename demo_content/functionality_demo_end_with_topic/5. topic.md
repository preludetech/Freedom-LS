A standard way to display tabular data using Markdown syntax.

**You do not need a special component tag; simply use pipes `|` and hyphens `-` to structure your data.**

## Syntax:

They support header rows. Defined by the first line of the table.

## How to use:

| Table Header 1| Table Header 2| Table Header 3|
|---|---|---|
| Cell 1| Cell 2| Cell 3|
| Cell 3| Cell 4| Cell 5|
| Cell 6| Cell 7| Cell 8|


## Cheat Sheet

```markdown
| Header | Header |
| --- | --- |
| Cell | Cell |
```
