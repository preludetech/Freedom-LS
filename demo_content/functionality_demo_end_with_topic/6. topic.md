A standard way to display tabular data using Markdown syntax.

**We support both *inline* code (for referencing variables in a sentence) and *block* code (for multi-line scripts).**

## Syntax:

They support syntax highlighting.

By specifying the language name (e.g., python, html, css) after the opening backticks.

## How to use:

### Inline Code

Use single backticks to highlight a short phrase. Example: Please check the `settings.py` file.

### Code Blocks

Use triple backticks for larger blocks of code.

```python

def hello_world():
    print("Hello, World!")

```
