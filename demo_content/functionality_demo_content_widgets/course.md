This course is two things at once: living documentation and a set of QA fixtures.

Every widget the content system ships with appears here, with every option exercised at least once. If you are authoring content, read it as a reference — each topic explains what a widget is for, then shows the exact markdown that produces it. If you are testing a rendering change, load this course and check that nothing breaks.

Three topics, grouped by what the widgets do: annotation and emphasis, media, and structured content.
