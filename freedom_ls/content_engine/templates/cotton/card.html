{% load content_tags %} {% comment %} Card component — a static content container with an optional header image, an optional title, and a markdown body. Attributes: src Optional. File path of a header image, looked up via get_file_by_path (the same path resolution c-picture uses). Omit for a card with no image. A header image is illustrative chrome — there is no lightbox; use c-picture for a zoomable figure. alt Alt text describing the header image for non-sighted users. Use alt="" for a purely decorative image. Only relevant when src is set. title Optional visible heading rendered above the body. size small | medium (default) | large. Sets the card's max width and the header image height. Cards are centred and deliberately narrower than the content column so they read as a self-contained panel. An unknown size falls back to medium. The body is the default slot, rendered as markdown. The header image is cropped to a fixed height with object-cover so cards of the same size line up regardless of the source image's aspect ratio. {% endcomment %} {% partialdef card_box %}
