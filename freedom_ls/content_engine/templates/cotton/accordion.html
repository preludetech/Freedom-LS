The markdown sanitiser collapses a bare `open` to `open=""`, so the default sentinel below lets the template distinguish "attribute present" (any value, including empty) from "attribute absent". Animation: The open/close transition uses a CSS grid-template-rows trick driven by the details[open] attribute selector. No JavaScript is needed — the browser toggles the open attribute natively, and CSS responds to it. The transition is gated behind @media (prefers-reduced-motion: no-preference) so users who prefer reduced motion see an instant reveal with no animation. Chevron: A CSS rotate driven by the details[open] selector flips the chevron icon when the disclosure opens. The rotation transition is also gated behind prefers-reduced-motion. Editorial note: Accordions are for elective depth — supplementary detail, optional context, or reference material that not every reader needs. Do not hide content that is required reading, forms part of assessed material, or that every learner needs to see.
