{% comment %} Single source of truth for the course-progress bar shown on course cards, the player TOC header, and the player breadcrumb chrome. The fill is always tinted to the course accent via `course-progress-`, so every progress bar matches its course's colours. Props: percentage - 0-100 value; drives the bar and the `{N}%` readout. accent_slot_key - course palette slot (e.g. "1") for the accent tint. label - aria-label for the bar. bar_class - bar thickness/shape utilities (default h-2). label_class - classes for the percentage readout span. suffix - text appended after the percentage (e.g. " complete"). Usage: {% endcomment %}
