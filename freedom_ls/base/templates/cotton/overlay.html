{% comment %} Shared accessible overlay built on the native element. Opened with showModal() (via the overlayDialog Alpine component) so it gets focus trapping, background inertness, Escape-to-close, and a ::backdrop scrim for free. Each consumer supplies its foreground content in the default slot and opens the dialog by calling open(triggerEl) on the overlayDialog component. Variants control the slide direction / presentation: - bottom-sheet: rises from the bottom edge (mobile TOC) - side-drawer: slides in from the left - lightbox: fade + scale from centre (picture zoom) Animation uses transform/opacity only (never height/top) to avoid layout jank. {% endcomment %}
